ܕܘܡܝ̈ܝܬܐ | A visualizer for Syriac word embeddings

Word embeddings are a set of computational techniques that are used to calculate which words behave similarly in context. They identify words which occupy the same semantic space, i.e. synonyms, but also antonyms and collocations.

To try it out using the data from Simtho (for words occurring at least twice), enter a Syriac word below (for example ܝܘܡܐ, to see what we mean by semantic space) and click "Go".

Please be patient; the data is large, the implementation inefficient. If a graph should appear, feel free to pan and zoom.
